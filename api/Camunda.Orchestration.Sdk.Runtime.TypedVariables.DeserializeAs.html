DeserializeAs<T>(object?, JsonSerializerOptions?)

Deserializes a Camunda variable or custom header payload to the specified type. Works on object properties that are JsonElement at runtime (the standard shape returned by the Camunda REST API).

public static T? DeserializeAs<T>(this object? payload, JsonSerializerOptions? options = null)

Parameters

payload object

The Variables or CustomHeaders property value from a Camunda API response.

options JsonSerializerOptions

Optional JSON serializer options. When null, uses default options with camelCase naming to match Camunda's JSON format.

Returns

T

The deserialized DTO, or default if the payload is null.

Type Parameters

T

The target DTO type to deserialize into.
